🏢 Site Subnets

Import per-site subnet definitions from Excel. Subnets are stored as site variables and used during template rendering.

Configured Sites

Import from Excel

Each row should represent one subnet. Multiple rows with the same site name are grouped together.

📋 Expected Excel Format

Required: site or site_name

Recommended: subnet in CIDR notation

Optional: gateway, vlan, description, or any site variable

site,subnet,gateway,vlan,description
Building A,192.168.1.0/24,192.168.1.1,10,Main office LAN
Building A,10.0.1.0/24,10.0.1.1,20,Guest Wi-Fi
